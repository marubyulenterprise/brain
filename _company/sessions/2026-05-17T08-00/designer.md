# ✨ Missing Edge Shock Component: 3가지 시각화 콘셉트 세트

## 공통 디자인 원칙
*   **타이포그래피:** `Pretendard` (기술적, 가독성 최우선). 제목/경고 문구는 `Bold`와 적절한 자간 조절을 통해 무게감을 부여합니다.
*   **컬러 팔레트:**
    *   ✅ **Success/Opportunity:** `#0A84FF` (Maru Tech Blue) — 명확하고 신뢰감 있는 색상.
    *   ⚠️ **Warning/Potential Value:** `#FFAA00` (Amber Alert) — 즉각적인 주의를 요하는 경고색.
    *   ❌ **Crisis/Loss:** `#C93838` (Crimson Red) — 재무적 손실을 상징하는 강렬한 색상.

---

### 💡 Concept 1: [The Glitch Shock] - 시스템 오류 경고형 (Technical Focus)
**주요 타겟 감정:** 불안감, 위협(System Integrity Threat).
**핵심 메커니즘:** 데이터가 '깨지는' 시각적 경험. 기술 전문가나 엔지니어에게 가장 큰 충격을 줍니다.

| 단계 | 시각 효과/애니메이션 | 색상 코드 | 내레이션 Hook (예시) |
| :--- | :--- | :--- | :--- |
| **1. 현재 상태** | 데이터 노드와 연결선이 완벽하게 구조화되어 보이지만, 일부 엣지(Edge)가 미세하게 깜빡거리며 불안정해 보인다. | `#E0E0E0` (Soft Gray) | "시스템은 정상 작동 중이라고 착각합니다." |
| **2. Missing Edge 발견** | 연결선이 갑자기 순간적으로 '글리치' 효과(Chromatic Aberration, 짧은 색상 분산 아티팩트)를 일으키며 끊어지거나 사라진다. 데이터 사일로 경계가 깨지는 듯한 시각적 노이즈 발생. | ⚠️ **`#FFAA00`** (Amber Alert) | "하지만 시스템 깊숙한 곳에, 연결된 채 버려진 '틈'이 있습니다." |
| **3. Loss/Crisis 상태** | 끊어진 Edge 부분 전체가 화면에서 빨간색 노이즈 필터(Red Noise Filter)를 입으며 격렬하게 깜빡인다 (Pulse & Flicker). 해당 사일로 경계에 `[ERROR: Missing Connection]`이라는 붉은 오류 코드가 오버레이된다. | ❌ **`#C93838`** (Crimson Red) | "이 연결 결핍은, 측정 가능한 기회비용 손실입니다." |
| **CTA 강조점:** | 빨간색으로 깨진 경계 위에 `[Predictive Indexing Run]` 버튼을 배치. 마우스 오버 시 글리치 효과가 다시 발생하며 '재활성화'를 암시한다. |

### 💰 Concept 2: [The Ledger Shock] - 재무 손실 그래프형 (Business/Financial Focus)
**주요 타겟 감정:** 손해에 대한 직접적인 두려움(Loss Aversion).
**핵심 메커니즘:** 명확한 숫자의 급락과 비교를 통한 충격. 사업가나 CFO에게 가장 직관적입니다.

| 단계 | 시각 효과/애니메이션 | 색상 코드 | 내레이션 Hook (예시) |
| :--- | :--- | :--- | :--- |
| **1. 현재 상태** | 세 개의 독립된 막대 그래프(데이터 사일로 A, B, C)가 나란히 배치되어 있으며, 각 그래프의 높이가 사용자의 '현재 인지 수익'을 나타낸다. | `#4CAF50` (Green - 안정적) | "당신이 현재까지 모은 데이터 자산 가치입니다." |
| **2. Missing Edge 발견** | 3개 사일로 간에 연결되어야 할 잠재적인 '통합 수익원' 영역(The Void)이 투명한 막대 그래프 형태로 나타난다. 이 영역은 아직 존재하지 않는 것처럼 보인다. | `#FFAA00` (Amber Alert) | "하지만 이 데이터들이 만난다면, 지금의 합계보다 훨씬 클 수 있습니다." |
| **3. Loss/Crisis 상태** | 가상의 '최대 잠재 수익(Max Potential Value)' 그래프가 솟아오르다가, Missing Edge 때문에 급격히 *꺾여서* 현재 수준에 머무르는 모습이 애니메이션으로 표현된다 (Loss Curve). 손실액 영역은 빨간색으로 명시된 `[Opportunity Cost: -₩XX,XXX]`로 표시된다. | ❌ **`#C93838`** (Crimson Red) | "당신은 지금 이 기회비용만큼의 돈을 벌지 못하고 있습니다." |
| **CTA 강조점:** | 빨간색 손실 구간 아래에 `[Calculate PVU & Bridge Gap]` 버튼 배치. 클릭 시, 꺾였던 그래프가 마루 블루로 되살아나며 수익이 상승하는 애니메이션(Bounce-up)을 보여준다. |

---

### ✨ Concept 3: [The Void Shock] - 스토리텔링 공백형 (Emotional/Narrative Focus)
**주요 타겟 감정:** 아쉬움, 잠재력에 대한 갈망(Aspiration).
**핵심 메커니즘:** 시각적 비유와 은유를 통해 '채워져야 할 공간'을 강조. 모든 사용자에게 가장 공감하기 쉬운 접근입니다.

| 단계 | 시각 효과/애니메이션 | 색상 코드 | 내레이션 Hook (예시) |
| :--- | :--- | :--- | :--- |
| **1. 현재 상태** | 데이터 요소(텍스트, 이미지, 아이콘 등)들이 무질서하게 쌓여있는 '창고' 같은 공간으로 표현된다. 각 요소는 흩어져 있지만, 그 자체로는 가치가 있어 보인다. | `#607D8B` (Cool Gray/Blue) | "당신은 엄청난 자원을 가지고 있습니다. 하지만…." |
| **2. Missing Edge 발견** | 데이터 요소들 사이의 공간(Void)에 미세한 빛줄기나 에너지 흐름이 감지되며, 이들이 서로 연결되어야 할 '최적 경로'가 시각화된다. (와이어프레임 느낌). | ⚠️ **`#FFAA00`** (Amber Alert) | "데이터는 그저 쌓여 있는 것이 아닙니다. 연결될 때 비로소 가치를 발산합니다." |
| **3. Loss/Crisis 상태** | 이 빛줄기(Potential Flow)가 막혀있는 구간이 어둠이나 안개 같은 '공백(Void)'으로 표현된다. 공백을 통과하려 할수록 에너지가 소멸되는 듯한 효과를 준다. | ❌ **`#C93838`** (Crimson Red) | "가장 가치 있는 흐름이, 단 하나의 연결고리로 인해 막혀버린 순간입니다." |
| **CTA 강조점:** | 공백(Void) 중앙에 `[Discover The Missing Edge]` 버튼 배치. 이 버튼을 누르면 어둠이 걷히며 빛줄기가 폭발적으로 연결되는 애니메이션으로 전환된다. |
