<create_file path="sessions/2026-05-18T04-30/LossDetectionGuarantee_Wireframe_V2.md"># ⚠️ Loss Detection Guarantee: 인터랙티브 위젯 스펙 (V2)
## Overview
이 섹션은 단순한 '보증' 문구가 아닌, 사용자가 스스로 자신의 비즈니스에 존재하는 잠재적 재무 손실($L_{Occ}$)을 인지하게 만드는 **인터랙티브 진단 도구**입니다. 기술적 권위와 위기감을 동시에 전달하는 것이 목표입니다.

## 📐 컴포넌트 구조: Loss Detection Engine (LDE)
### A. 헤더 섹션 (Authority & Urgency)
*   **헤드라인:** "당신의 비즈니스는 지금, 눈에 보이지 않는 손실을 겪고 있습니다."
*   **서브 카피:** "Loss Detection Engine(LDE)는 시스템의 원자적 트랜잭션 기반으로 미인식 재무 리스크를 정밀 진단합니다. (데이터 분석가/엔지니어용)"
*   **공신력 마크 위치 (Seal):** 섹션 상단 우측에 배치. "Audit-Grade Security Certified"와 같은 문구와 함께 '검증된 시스템' 느낌의 엠블럼을 사용하며, 미세한 진동 애니메이션(Pulse)을 부여하여 신뢰도를 높임.

### B. 인터랙티브 위젯 영역 (The Diagnostic Engine)
**1. Loss Vector Input (체크박스 그룹)**
*   사용자에게 다음 3~5가지의 '잠재적 손실 벡터'를 제시합니다. (예시만 사용자가 실제로 체크하게 만듭니다.)
    *   ☐ 데이터 파편화(Data Siloing): 여러 시스템에 분산된 정보로 인한 비일관성.
    *   ☐ 프로토콜 미준수(Protocol Drift): 최신 업계 표준 또는 기술 스펙을 따르지 못하는 지연 리스크.
    *   ☐ 실시간 검증 부재(Lack of Real-time Validation): 잠재적 오류를 즉시 감지하지 못하고 누적되는 위험.
*   **핵심 로직:** 사용자가 체크박스를 클릭할 때마다, 시스템은 해당 벡터에 할당된 가중치($W$)를 리스크 점수 합산에 반영합니다.

**2. 실시간 리스크 대시보드 (The Core)**
이 영역은 가장 중요하며, 세 가지 하위 컴포넌트로 구성됩니다.
*   **[① Loss Score Meter]**
    *   **표현:** 원형 게이지(Gauge) 또는 그래프 형태의 시각화 위젯.
    *   **범위:** 0점 (안전) $\to$ 100점 (Critical Risk).
    *   **색상 변화:** 🟢 (Safe/Green) $\to$ 🟡 (Warning/Amber) $\to$ 🔴 (**CRITICAL/RED**)
    *   **애니메이션:** 점수가 올라갈수록 게이지가 '빨리' 채워지면서, 경고음과 함께 시각적 충격(Flicker)을 주어야 합니다.
*   **[② 리스크 상세 분석 (Loss Breakdown)]**
    *   사용자가 체크한 각 벡터별로, 그 점수와 함께 "이것이 놓치는 재무 손실액 예측: $L_{Occ} = X$원"이라는 문구를 명시적으로 보여줍니다.
*   **[③ Total Loss Prediction Box]** (최종 결과)
    *   위젯의 중앙에 가장 크고 굵은 글씨로 표시됩니다.
    *   **포맷:** **미인식 기회 손실 예측($L_{Occ}$): [Critical Red 배경] $X$원 ~ $Y$원**
    *   이 숫자는 사용자가 다음 단계(CTA)를 누르도록 강제하는 핵심 트리거입니다.

### C. CTA (Action & Conversion)
*   **버튼 텍스트:** "Loss Detection Engine으로 나의 시스템 진단 받기" 또는 "Pro Tier로 $L_{Occ}$ 방어막 구축하기"
*   **위치:** 리스크 점수 대시보드 바로 아래, 시각적 무게감이 가장 무거워야 합니다.
*   **상태 변화:** 사용자가 버튼을 누르면, '진단 요청 완료' 메시지와 함께 다음 단계(Pricing/Onboarding)로 부드럽게 스크롤되어 이동합니다.
