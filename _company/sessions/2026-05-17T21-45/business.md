### 💰 티어별 가격 책정 근거 및 로직 상세 (Developer 가이드)

#### 1. Free Tier: The Hook (Pain Point Amplifier)
*   **가격:** $0 (무료)
*   **제한 API/로직 구현 지침:**
    *   `KB_Extraction`: `input_source_type == [Text, SinglePDF]` 일 경우만 성공 응답. `source_length > 500` 이면 에러 발생 및 "배치 크기 제한(Max 500자)" 메시지 출력.
    *   `L_Amount_Engine`: API 호출은 가능하나, 계산 결과에 **"추가 기능을 통해 원천 데이터를 연결하면 더 정확합니다."**라는 워터마크/경고 문구를 강제로 삽입하여 불완전함을 인지시킴.
    *   `Workflow_Automation`: 모든 자동화 기능의 실행 전에 반드시 `ManualConfirmationRequired: True` 플래그를 띄워 수동 개입을 의무화함.

#### 2. Starter Kit: The Utility (Scale Barrier)
*   **가격:** 월 $39~$49 (가정)
*   **제한 API/로직 구현 지침:**
    *   `KB_Extraction`: `input_source_type == [PDF, DOCX]` 인 경우 성공. 최대 10건까지 처리 허용.
    *   `L_Amount_Engine`: 계산 결과에 **"데이터 통합 분석(Pro 전용)을 통해 예측 정확도를 30% 이상 향상시킬 수 있습니다."**라는 제안 배너를 노출함. (사용자가 부족함을 느끼게 함)
    *   `Workflow_Automation`: `MaxTriggerCount: 1/Month`. 이 제한에 도달하면 "월별 트랜잭션 한도 초과" 오류와 함께 Pro 유도 CTA가 발동해야 함.

#### 3. Pro Tier: The Asset Builder (Maximum Value Unlock)
*   **가격:** 월 $99~$149 (가정)
*   **핵심 API/로직 구현 지침:**
    *   `KB_Extraction`: `input_source_type == [API_Gateway]` 일 경우만 성공. **무제한 처리.** 이 로직이 가장 가치 높은 기능을 제공하며, Free/Starter에서 경험할 수 없는 '연결의 자유'를 상징해야 함.
    *   `L_Amount_Engine`: API 호출 시 **최종 보고서(PDF/CSV)** 다운로드를 가능하게 하고, 이 보고서는 다른 모든 티어에서는 불가능한 고유 자산으로 취급되어야 함.
    *   **추가 핵심 로직: `AtomicTransactionGuaranteed()`:** 결제 시스템과 연동되는 모든 API 호출에 대해 원자적 트랜잭션 보장 플래그를 적용하여, 금융/비즈니스 안정성을 최고 수준으로 끌어올려 신뢰도를 극대화함.
