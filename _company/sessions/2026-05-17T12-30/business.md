## 📊 SaaS MVP 수익화 모델 최종 제안 (Pricing Model)

**핵심 가치 포지셔닝:** "당신의 콘텐츠 속에 숨겨진 돈이 되는 데이터의 파편(Missing Edge Shock Component)을 찾아내고, 이를 구조화된 자산으로 전환합니다."

### 💡 1. Freemium Tier: Starter ($0/월)
가장 낮은 진입 장벽을 통해 대규모 사용자 풀(Pool)을 확보하고, 핵심 가치인 '충격' 자체를 경험하게 만드는 것이 목표입니다. 초기 사용자 데이터를 수집하여 시장 적합성을 검증합니다.

| 구분 | 내용 (Feature Set) | 제한 사항 (Limitation) |
| :--- | :--- | :--- |
| **핵심 기능** | 콘텐츠 재가공, 기본적인 Shock Component 추출(최대 3개까지), 기본 데이터 구조화 변환. | 월별 원천 콘텐츠 업로드 제한 (예: 5개/월). 분석 가능한 총 페이지 수 제한 (예: 10,000 단어). |
| **KPI 목표** | **CAC(고객 확보 비용) 최소화 / 액티브 사용자 수 극대화.** 초기 바이럴 및 입소문. | 사용자가 '이 정도로는 부족하다'는 인식을 갖도록 설계하는 것이 핵심입니다. |
| **도입 시점** | MVP 런칭 (Q2 초반) — 시장 검증 단계에서 최대 규모 확보용. |

### 💪 2. Paid Tier: Professional ($49 ~ $99/월, 예상 범위)
프리랜서나 소규모 팀이 본업과 병행하며 실질적인 수익을 창출할 수 있는 핵심 구간입니다. '시간 절약'의 가치와 '안정성(Reliability)'에 초점을 맞춥니다.

| 구분 | 내용 (Feature Set) | 제한 사항 (Limitation) |
| :--- | :--- | :--- |
| **핵심 기능** | 모든 Starter 기능 포함 + 무제한 원천 콘텐츠 업로드, 고급 Shock Component 분석(Top 10개까지), 자동화된 지식 베이스 구축 API 연동. | 사용량 상한선을 설정하여 다음 단계로 유도 (예: 월 처리 트랜잭션 수 500건 제한). 전담 고객 지원이 필요할 경우 추가 비용 발생 가능성 암시. |
| **KPI 목표** | **MRR(월 반복 매출) 확보 / LTV(고객 생애 가치) 증대.** 가장 높은 전환율을 기대하는 구간입니다. |
| **도입 시점** | MVP 출시 후 1~2개월 (Q2 중반) — 첫 유료 고객 발생 단계. |

### ✨ 3. Premium Tier: Enterprise ($499+ /월, 협의)
대형 에이전시나 콘텐츠를 주력으로 하는 회사에 적합합니다. 단순한 도구를 넘어, 비즈니스 프로세스 자체에 통합되는 '솔루션'으로서 포지셔닝해야 합니다.

| 구분 | 내용 (Feature Set) | 제한 사항 (Limitation) |
| :--- | :--- | :--- |
| **핵심 기능** | 모든 Professional 기능 포함 + 전용 API Gateway, 커스텀 Shock Component 로직 개발(맞춤화), 팀원 수 무제한 관리, SLA 기반의 최고 수준 안정성 보장. | 직접적인 가격 제한보다는 '전담 컨설팅 및 통합 비용'으로 접근하는 것이 적절합니다. |
| **KPI 목표** | **ARPU(사용자당 평균 매출) 극대화 / 성공 사례 확보.** 레퍼런스 구축 목적. |
| **도입 시점** | 베타 테스트 완료 후, 초기 고객사 1~2곳 발굴 (Q3 초반). |
