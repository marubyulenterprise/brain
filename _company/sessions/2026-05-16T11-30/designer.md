### 🚀 [최종 산출물] PVU 기반 CTA 인터랙션 디자인 스펙 및 마이크로카피 정의서

**목표:** 사용자가 PVU(잠재 추가 순이익) 수치를 보고 즉각적인 행동을 하도록 유도하는 '골드 경고' CTA의 클릭 전/후 경험을 상세히 설계한다. (단순 버튼 디자인 $\rightarrow$ 강력한 액션 플로우로 전환)

**참여 에이전트:** Designer (리딩), Writer (카피라이팅 강화)

#### 1. UI/UX 시나리오 개요: '골드 경고' 상태
*   **상태 정의:** 시스템이 분석 결과, 사용자가 현재 놓치고 있는 잠재적 수익 기회(PVU)를 감지했을 때.
*   **핵심 메시지 전달 방식:** 단순한 "놓쳤습니다"가 아니라, **"[경고] OOO라는 Edge 발견! 예상 PVU: ₩XX만원 (잠재 순수익)"** 형태로 정량적 손실 기회를 제시한다.

#### 2. CTA 디자인 및 기능 상세 정의
| 항목 | 내용 | 사양/디자인 지침 | 비고 |
| :--- | :--- | :--- | :--- |
| **위치 (Placement)** | 메인 대시보드 중앙, PVU 수치 바로 하단에 배치. 경고 영역 전체를 감싸는 형태로 시각적 집중도를 높임. | 배경 색상: 밝은 골드(Gold Alert) 오버레이 사용. Pulse & Flicker 애니메이션을 적용하여 지속적인 주의 환기 필요. | 가장 중요한 시각 요소. |
| **CTA 버튼 텍스트** | `▶️ 지금 바로 기회 활용하기 (PVU 분석 실행)` | 단순 '클릭' 대신 '행동(Action)' 동사를 사용하여 즉시성을 강조. | 사용자가 무엇을 얻는지 명확히 제시해야 함. |
| **호버/포커스 상태** | 버튼 위로 마우스를 올리거나 포커싱 시, PVU가 계산되는 핵심 변수 3가지($A \times B \times C$)를 작은 애니메이션으로 재배열하며 '복잡한 로직'의 느낌을 부여. | Hover Text: "놓치면 아까운 잠재적 가치를 즉시 측정합니다." | 신뢰도와 전문성을 강화하는 장치. |
| **클릭 액션 (핵심)** | 버튼 클릭 시, 새로운 페이지로 이동하는 것이 아니라, **'PVU 상세 진단 모달(Modal)'**이 부드럽게 오버레이 되어 나타나야 한다. | 트랜지션: 0.3초의 Fade-in 효과. Modal 크기는 화면 중앙에 가장 크게 배치하여 집중 유도. | 사용자 이탈 방지 및 맥락 유지. |

#### 3. Microcopy & UX Flow 정의 (Writer 협업 영역)
CTA 클릭 후 나타나는 **'PVU 상세 진단 모달'** 내부의 카피라이팅과 흐름을 구체적으로 정의합니다.

**A. 모달 헤드라인 (H1)**
*   **내용:** "💰 [발견] 놓치고 있던 OOO 잠재 순수익 기회 분석 보고서"
*   **효과:** 사용자가 '문제 해결'이 아닌, **'숨겨진 돈 찾기'**라는 긍정적 경험을 하게 함.

**B. 모달 본문 (Body Copy)**
*   **톤:** 매우 전문적이지만, 사용자에게는 쉬운 비유를 사용해야 함.
*   **내용 구조:**
    1.  **문제 정의 (Pain):** "현재 데이터 구조만으로는 이 연결고리(Edge)가 존재함을 파악하기 어렵습니다."
    2.  **솔루션 제시 (Discovery):** "하지만 저희 AI 분석 엔진은 [A]와 [B]라는 두 개념이 결합될 때 발생하는 '숨겨진 가치'를 포착했습니다."
    3.  **구체적 액션 플랜 (CTA Context):** 이 가치를 현실화하려면, 사용자의 **[다음 단계의 행동/콘텐츠 기획]**이 필수입니다.

**C. 모달 내 최종 CTA 버튼**
*   **텍스트:** `✅ [액션 실행] 발견된 Edge를 기반으로 콘텐츠 플랜 생성하기`
*   **기능:** 이 버튼을 누르면, 사용자는 'PVU가 계산된 데이터 셋'을 바탕으로 **콘텐츠 기획(Writer 영역)**이나 **워크플로우 다이어그램 수정(Secretary 영역)** 화면으로 자연스럽게 연결되어야 한다.
